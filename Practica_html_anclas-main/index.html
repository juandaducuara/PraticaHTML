A

Archivo: Un archivo es un elemento que almacena un conjunto de datos para su posterior uso.

Ancla: Vinculo que hace referencia a un lugar o una pocicion dentro de la misma pagina web. Parecido al Link.

Antivirus: Programas especializados en la detección y, si es posible, en la destrucción de virus informáticos.


B

Backup: Copia de Respaldo o Seguridad.

Bit: Es la unidad de información más pequeña. Puede tener sólo dos valores o estados: 0 o 1, encendido o apagado.

Buscador: Los buscadores son sitios Web, especializados en localizar información a través de Internet.


C

C: El lenguaje C es una herramienta de programación de tipo general, utilizada para el desarrollo del sistema operativo Unix.

Cable: Conexión a Internet de alta velocidad que se hace a través de fibra óptica.

Chip: Utilizado habitualmente como sinónimo de procesador, se trata de una oblea de silicio sobre la que se imprime un microcircuito.


D

Dato: Unidad mínima de información, sin sentido en sí misma, pero que adquiere significado en conjunción con otras precedentes de la aplicación que las creó.

Deletear: Barbarismo procedente de la palabra inglesa delete, que significa borrar.

Digito: Carácter simple en un sistema de numeración.


E

Editor: Un programa que puede crar o modificar archivos de un tipo determinado.

Encapsulación: En relación a la POO, consiste en mantener datos y métodos dentro de una única abstracción.

Enlace: Es una palabra, una frase o un gráfico de un hipertexto que conecta con otra información.


F

Fax: Método y aparato de transmisión y recepción de documentos mediante la red telefónica conmutada.

Fibra óptica: Sistema de transmisión que utiliza fibra de vidrio como conductor de frecuencias de luz visible o infrarrojas.

Formulario: Objeto de una base de datos utilizado para la introducción, edición y gestión de la información de una tabla de la base de datos.


G

Gateway: Computador que realiza la conversión de protocolos entre diferentes tipos de redes o aplicaciones.

Gigabyte: Mil millones de bytes. También GB, Gbyte y G-byte.

Google: Buscador de páginas web en Internet (y el más popular por el momento).


H

Hacking: Es la penetración directa a un sistema informático.

Hardware: Conjunto de componentes materiales de un sistema informático.

HTML: Es el lenguaje estándar para describir el contenido y la apariencia de las páginas en el WWW.


I

IDS: (Intrusion Detection System). Sistema de detección de intrusos.

Información: Elemento fundamental que manejan los ordenadores en forma de datos binarios.

Intel: Fabricante de los procesadores que llevan su nombre, inventó la arquitectura 80X86.


J

JACK: Tipo de conexión que utilizan muchos cables, por ejemplo los de algunos dispositivos de sonido.

Java: Lenguaje de programación orientado a objetos para la elaboración de aplicaciones exportables a la red.

Jpeg: Formato gráfico con compresión con perdidas que consigue elevados ratios de compresión.


K

Kb: Kilobyte. Unidad de medida utilizada en informática que equivale a 1.024 bytes.

Keylogger: Programa que recoge las pulsaciones que realiza el usuario sobre el teclado

KISS: (Keep it simple, stupid). Término para definir la simplicidad de la programación.


L

LAN: Red de área local. Define la conexión física y lógica de ordenadores en un entorno.

Llaves: Concepto de datos de un registro que se usa para identificarlo.

Login: Es el proceso que controla el acceso individual a un sistema mediante la identificación del usuario.


M

Malware: Son todos aquellos programas diseñados para causar daños al hardware, software, etc.

Megabyte: Nombre de una unidad de información equivalente a un millón de bytes

Multimedia: Alude a la combinación en un ordenador de sonido, gráficos, animación y vídeo.
